# StockRail: Crypto Rails for Real Stock Investing

**Crypto in. Real equities out.**

StockRail is an institutional-grade financial infrastructure layer that connects global crypto liquidity to regulated public equity markets. We provide the compliant "bridge" that allows digital asset holders to invest in real shares through licensed brokers, bypassing the friction of legacy fiat-native banking.

## 🚀 The Vision
To become the default infrastructure rail for the global crypto economy, unlocking $100T+ in public equities for 400M+ crypto users. We focus on **market access and funding rails**, not synthetic assets.

## 🛠 Core Pillars
- **Real Ownership:** No tokenized stocks or synthetics. Users own real shares held at licensed brokers.
- **Atomic Settlement:** Instant conversion from crypto (USDT/USDC/ETH/BTC) to fiat for immediate equity execution.
- **Compliance-First:** Built-in KYC/AML, MPC custody, and direct integration with regulated brokerage APIs.
- **Global Reach:** Removing cross-border funding barriers for emerging markets and crypto-native investors.

## 🏗 Project Structure
- `/src`: React (TypeScript) + Vite frontend landing page and dashboard.
- `/services` (Planned): Microservices for Ledger, Execution, and Liquidity Routing.
- `/docs`: Technical specifications and regulatory mapping.

## 🚦 Getting Started (Frontend)

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
